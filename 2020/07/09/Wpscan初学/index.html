Wpscan初学

Posted on 2020-07-09 In 安装软件 , 小试牛刀

WPScan是Kali Linux默认自带的一款漏洞扫描工具，能够扫描WordPress网站中的多种安全漏洞，其中包括WordPress本身的漏洞、插件漏洞和主题漏洞。

1.注意：现在使用wpscan需要使用官方的api-token，这个可以自己注册一个免费的账号。

如果不使用的话扫描结果是不会输出漏洞信息的！！

注册地址：https://wpvulndb.com/users/choose_plan

注册完成后可以选择每天免费的50api

虽然kali里自带wpscan但是想要在自己其他的设备上安装的话可以用下面的命令

git clone https://github.com/wpscanteam/wpscan.git

2.查看帮助信息

wpscan -h

3.更新漏洞库

wpscan --updata

4.扫描网站

wpscan --url 网站的URL -e --api-token xxxxxxxxxxxxxxxxxxx #-e简单进行整体快速扫描

发现的漏洞

注意：这里扫描到的是站点的一些信息并且列出可能是漏洞的地方。注意，这里wpscan判断是否有漏洞，是根据wordpress的版本判定的，只要你的版本低于存在漏洞的版本，那么，它就认为存在漏洞，所以，这个没有太多的参考性。

5.简单扫描wordpress的插件(plugin)

wpscan --url 网站的URL --enumerate p --api-token xxxxxxxxxxxxxxxxx

6.完整扫描wordpress的插件(plugin)

wpscan --url 网站的URL --enumerate ap

7.列举wordpress的用户

wpscan --url 网站的URL --enumerate u

8.扫描使用的主题和漏洞

wpsan --url 网站的URL --enumerate vt

9.用字典暴力破解

wpscan --url 网站的URL -P /user/share/wordlist/rockyou.txt -U admin -t 100 #指定用户名为admin，密码为 /user/share/wordlist/rockyou.txt 字典文件中的数据 设置线程100
