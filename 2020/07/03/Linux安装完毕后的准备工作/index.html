Linux安装完毕后的准备工作

Posted on 2020-07-03 Edited on 2020-07-09 In 系统准备 , 基础奠基

Linux系统是很多想进一步学习计算机的人所必须要了解的，但在系统安装完成后却不是所有的准备工作都完成了，相反我们还有一些工作要做。

Centos7

1.更改源

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo_bak

wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

yum makecache

yum -y update

Kali

1.更改root密码

sudo passwd root

2.更改源

vim /etc/apt/sources.list

apt-get update

apt-get upgrade

apt-get dist-upgrade

reboot

3.端口开启

apt-get install ufw（下载防火墙）

ufw default deny（默认禁止访问所有 ）

ufw allow 22/tcp（开放22/TCP端口）

ufw allow 53（开放53端口(tcp/udp）

ufw enable（启动ufw防火墙）

ufw disable（关闭ufw防火墙）

ufw status numbered（查看规则，显示行号）

4.ssh设置为开机启动

vim /etc/ssh/sshd_config(修改ssh_config文件)

将#PasswordAuthentication no的注释去掉，并且将NO修改为YES //我的kali中默认是yes

将#PermitRootLogin yes的注释去掉 //我的kali中默认去掉了注释

/etc/init.d/ssh start(启动SSH服务)

/etc/init.d/ssh status(验证SSH服务状态)

update-rc.d ssh enable(添加开机自启动)

5.安装中文输入法

apt-get install fcitx-table-wbpy fonts-wqy-microhei fonts-wqy-zenhei -y(安装小企鹅中文输入法)

fcitx(启动输入法)

重启
